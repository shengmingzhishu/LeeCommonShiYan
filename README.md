# LeeCommonShiYan
灵力检测实验室


开发一个在线体检的小程序，支持体检商品展示、购物车、订单、支付功能，购买的套餐支持填写连接系人，支持商户发货功能，展示物流进度，用户收到采样设备后，可以自行邮寄样品到公司，也可以选择由公司上门取样功能， 取样完成后，支持公司后台上传检验报告，word pdf等。每个用户注册可以填写很多采样人信息； 所以这是一个展示检测套餐，可购买套餐，选择检测设备收货后自己寄送到公司，还是预约公司人员来上面取样，支持公司上传多次的检测结果。支持支付功能的小程序，同时支持多不同地域，选择就近城市的后台查看下单数据；如果所在成功未指定具体商户管理员，则使用系统最大的管理员； 所以这个小程序有用户侧看到的应用，也有后端管理侧的应用系统，这个管理系统是公司侧的。 所有的下单数据都要记录城市与所在城市管理的公司；这样方便利益分成；后期可能同一个城市有多个公司； 这个系统后端默认创建一个超级管理公司，并指定超级管理员账号 root 1234，其他城市公司创建都又这个超级管理员公司账号完成。
